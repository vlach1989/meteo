# Meteo App

This is a simple Next.js application to display the current temperature.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the
result.

## Environment Variables

The application requires the following environment variable to be set in a
`.env.local` file:

- `API_NOW`: The URL to the CSV file containing the current temperature data.

Example `.env.local` file:

```
API_NOW="your-api-url-here"
```
